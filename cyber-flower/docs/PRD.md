## 1. 产品概述

### 1.1 一句话定义

> AI驱动的真实感虚拟养花微信小程序——每一朵花都有AI赋予的独特外观、生长逻辑和对话人格。

### 1.2 核心价值主张

| 价值维度 | 描述 |
|----------|------|
| **情感陪伴** | AI对话让花拥有"人格"，形成双向情感连接 |
| **独一无二** | AI生成每朵花的DNA，不存在两朵相同的花 |
| **节气美学** | 跟随二十四节气，真实感渲染，花与自然同频 |
| **社交连接** | 微信生态天然社交，互赠花朵、花园互访 |

### 1.3 产品定位

```
                      AI驱动 ↑
                            |
         禅意养成 ← ——— + ——— → 游戏化
                            |
                     传统绿植 ↓
```

赛博养花位于四个象限的交汇点：有AI驱动的独特性、禅意养成的放松感、游戏化的收集乐趣、以及真实绿植的视觉亲近感。

### 1.4 平台策略

- **首发平台**: 微信小程序
- **优势**: 即用即走、社交传播自然、支付体系成熟、云开发降低运维成本
- **后续扩展**: Web H5版本、APP（React Native）

## 2. 用户画像

### 2.1 核心用户

| 画像 | 描述 | 核心诉求 | 使用场景 |
|------|------|----------|----------|
| **都市白领小美** | 25-35岁女性，办公室工作，喜欢植物但养不活 | 低维护的养成体验、情感陪伴 | 通勤/午休时照料、睡前与花对话 |
| **学生小绿** | 18-24岁，喜欢新鲜事物，社交活跃 | 收集稀有品种、与好友互动 | 课间打卡、朋友圈分享 |
| **文艺中年阿树** | 35-50岁，热爱园艺，对传统文化有感情 | 节气文化、深度养成 | 清晨照料花园、记录生长日记 |

### 2.2 用户场景

#### 场景A：日常打卡
```
早晨 8:00 — 小美打开微信 → 进入小程序 → 浇水、查看生长状态
→ AI花说"早安，今天的阳光很好呢" → 获得今日成长值 → 关闭
耗时: 2-3分钟
```

#### 场景B：获得新品种
```
周末下午 — 小绿输入关键词"雨后彩虹" → AI生成一颗彩虹基因种子
→ 种下后等待3天发芽 → 每天拍照记录 → 开花后分享到朋友圈
→ 好友评论"好美！求种子！"
```

#### 场景C：节气事件
```
立冬 — 花园飘起初雪 → 花进入"冬藏"状态 → 形态转为含蓄内敛
→ AI提示"冬天来了，我需要少一点水" → 用户调整养护策略
```

### 3.2 功能详细规格

#### F01 — AI关键词生成种子

**描述**: 用户输入文字描述（心情、意象、关键词），AI解析语义并生成一颗独一无二的花种。

**输入**:
- 文字: 1-50字的关键词/描述
- 可选: 心情标签（开心/平静/忧伤/兴奋/...）

**AI处理流程**:
```
用户输入 → 语义解析 → 生成花种DNA → 生成预览图 → 命名建议
```

**花种DNA结构**:
```json
{
  "seedId": "s_xxx",
  "createdAt": "2026-06-13T08:00:00Z",
  "origin": {
    "type": "keyword",
    "input": "雨后彩虹",
    "parsedMood": "清新",
    "parsedImagery": ["雨", "彩虹", "天空", "水滴"]
  },
  "genome": {
    "species": "iris_variant_7",
    "petalColor": { "r": 235, "g": 178, "b": 210 },
    "petalShape": "layered_round",
    "stemLength": 0.7,
    "leafPattern": "droplet_edge",
    "bloomSize": 0.65,
    "rarity": "rare"
  },
  "growthProfile": {
    "germinationDays": 3,
    "bloomDays": 14,
    "fullLifeDays": 45,
    "seasonPreference": "spring",
    "waterNeed": 0.6,
    "lightNeed": 0.8,
    "fertilizerNeed": 0.4
  },
  "personality": {
    "name": "虹雨",
    "tone": "温柔而梦幻",
    "speakingStyle": "诗意、偶尔引用诗词",
    "moodBaseline": 0.7
  }
}
```

**输出**:
- 种子卡片（含预览图）
- AI生成的名字
- 稀有度标签
- 种植按钮

**限制**:
- 每日免费生成: 1次
- 上限: 每月30次

#### F03 — 养成系统

**交互动作**:

| 动作 | 频率 | 效果 | 可视化反馈 |
|------|------|------|------------|
| 💧 浇水 | 1-2次/天 | 水分+10~20 | 水滴动画、土壤颜色变深 |
| 🌱 施肥 | 1次/3天 | 养分+15~30 | 小颗粒洒落、叶片光泽增强 |
| ☀️ 光照 | 调整设置 | 影响光合作用速度 | 光晕效果、花朝向光源 |
| ✂️ 修剪 | 按需 | 影响形态、促进分枝 | 枝叶飘落、新芽萌出 |

**生长阶段**:

```
种子期(0-3天) → 发芽期(3-7天) → 生长期(7-14天) → 花苞期(14-18天) 
→ 盛花期(18-35天) → 结果期(35-45天) → 休眠期 → 轮回
```

**AI生长引擎**:
- 不是固定时间线，AI根据用户照料质量和频率动态调整生长速度
- 同一品种不同用户养出来的花会有形态差异
- 极端情况：长期不照料 → 花进入"野生状态"，形态更加野性

#### F07 — AI对话陪伴

**对话系统设计**:

| 层级 | 功能 | 描述 |
|------|------|------|
| L1 | 状态播报 | "我今天喝饱水了，叶子在舒展开呢" |
| L2 | 情绪反馈 | 根据照料情况表达开心/失落/期待 |
| L3 | 知识问答 | 回答关于植物/节气/自然的问题 |
| L4 | 深度对话 | 记住用户说过的话，形成长期记忆 |
| L5 | 主动关心 | 检测到用户情绪（通过文字分析），主动安慰 |

**对话触发**:
- 用户主动进入对话页
- 每日首次进入花园时花主动问候
- 特殊事件触发（开花/枯萎/节气更替）

**人格模型**:
```
种子DNA决定基础人格 → 用户互动塑造人格演变 → 形成独一无二的"花格"
```

## 4. 用户流程

### 4.1 首次使用流程

```
打开小程序 → 微信授权登录 → 创建昵称/头像 
→ 新手引导: "输入一个词，让AI为你生成第一颗种子"
→ 输入关键词 → AI生成种子 → 种下第一朵花
→ 引导浇水 → 花苗出现 → 引导对话 → 完成新手教程
→ 进入个人花园主页
```

### 4.2 日常使用流程

```
进入花园 → 花主动问候 → 查看生长状态 → 执行照料动作
→ (可选)与花对话 → (可选)访问好友花园 → (可选)查看图鉴
→ 获得今日成长值 → 退出
```

### 4.3 社交流程

```
好友分享链接 → 进入好友花园 → 观赏好友的花
→ (可选)点赞 → (可选)留言 → (可选)赠送种子
→ 返回自己的花园
```

## 5. 交互设计

### 5.1 页面层级结构

```
├── 启动页 (Splash)
├── 主页 — 花园视图 (Home Garden)
│   ├── 照料面板 (Care Panel)
│   ├── 对话气泡 (Chat Bubble)
│   └── 种子背包 (Seed Inventory)
├── 育种页 (Breed)
│   ├── 关键词输入
│   ├── 每日领取
│   └── 种子详情
├── 对话页 (Chat)
│   ├── 聊天界面
│   └── 情绪展示
├── 图鉴页 (Collection)
│   ├── 品种列表
│   └── 花朵详情
├── 社交页 (Social)
│   ├── 好友列表
│   └── 好友花园
└── 个人中心 (Profile)
    ├── 设置
    ├── 纪念花园
    └── 关于
```

### 5.2 核心交互原则

1. **触感优先**: 浇水、施肥等操作要有真实的物理反馈（振动、动画）
2. **渐进披露**: 新手期简化界面，逐步解锁复杂度
3. **情感化微交互**: 花的微表情、摆动、落叶等细节
4. **无障碍**: 支持大字模式、简化操作

## 6. 非功能需求

### 6.1 性能要求

| 指标 | 目标值 |
|------|--------|
| 首屏加载 | < 2秒 (4G网络) |
| 页面切换 | < 300ms |
| 花渲染帧率 | ≥ 30fps |
| AI种子生成 | < 15秒 |
| AI对话响应 | < 3秒 |
| 小程序包大小 | 主包 < 2MB, 总包 < 20MB |

### 6.2 兼容性

- 微信版本 ≥ 8.0
- iOS ≥ 14, Android ≥ 8.0
- 屏幕适配: 320px ~ 428px 宽度

### 6.3 安全与隐私

- 用户对话数据加密存储
- AI生成内容过滤（防止不当内容）
- 符合微信小程序隐私政策
- 用户数据可导出/删除
