## 1. 设计哲学

### 1.1 核心设计原则

> **"让数字生命在真实中呼吸"**

| 原则 | 描述 | 实践 |
|------|------|------|
| **真实至上** | 花朵是主角，UI是舞台，不抢戏 | 大面积留白、花朵居中、UI元素半透明 |
| **温柔科技** | 科技感隐藏在细节中，不冰冷 | 微妙的发光、细腻的过渡、精致的数字细节 |
| **节气韵律** | 设计随季节呼吸，色彩/氛围应季而变 | 四季配色方案、节气微动效 |
| **触感第一** | 每一次交互都有真实的物理反馈 | 振动、弹性动画、粒子特效 |
| **渐进深度** | 新手简单，老手有深度可探索 | 功能分级解锁、手势捷径 |

### 1.2 设计参考

```
视觉参考系:
┌─────────────────────────────────────────────┐
│  真实植物摄影  +  柔和光影  +  微科技细节    │
│  (如PlantSnap)   (如Muji)    (如iOS健康)    │
│                                             │
│  整体调性: 温暖 × 精致 × 有生命感            │
└─────────────────────────────────────────────┘
```

## 3. 字体系统

### 3.1 字体选择

```
中文字体:
  标题: 系统默认衬线体 (Songti SC / Noto Serif CJK)
        → 优雅、有文化底蕴，适合植物主题
  正文: 系统默认无衬线 (PingFang SC / Noto Sans CJK)
        → 清晰、现代，可读性好

英文/数字:
  UI:    SF Pro Text / Roboto
  特殊:  Playfair Display (花名、装饰性文字)

小程序字体回退:
  font-family: "Songti SC", "Noto Serif CJK SC", "STSong", 
               -apple-system, "PingFang SC", sans-serif;
```

### 3.2 字号层级

| 层级 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| H1 | 36rpx | 1.3 | Bold 700 | 花名、大标题 |
| H2 | 30rpx | 1.3 | Semibold 600 | 页面标题 |
| H3 | 26rpx | 1.4 | Medium 500 | 模块标题 |
| H4 | 22rpx | 1.4 | Medium 500 | 卡片标题 |
| Body-L | 18rpx | 1.6 | Regular 400 | 正文大 |
| Body | 16rpx | 1.6 | Regular 400 | 正文 |
| Body-S | 14rpx | 1.5 | Regular 400 | 辅助说明 |
| Caption | 12rpx | 1.4 | Regular 400 | 标注/时间戳 |
| Number | 40rpx | 1.2 | Bold 700 | 数据展示 |

> 注: 微信小程序使用 `rpx` 单位 (1rpx = 屏幕宽度/750)

## 4. 图标系统

### 4.1 风格定义

```
图标风格: 线性 + 微填充
  - 线条宽度: 1.5px (细线) / 2px (粗线)
  - 圆角: 圆润 (borderRadius: 4rpx)
  - 风格: 有机线条，避免机械直角
  - 参考: Feather Icons 的自然版
```

### 4.2 核心图标

```
功能性图标 (24×24 rpx):
  💧 浇水 — 水滴形状，带涟漪
  🌱 施肥 — 颗粒洒落
```

## 7. 动效规范

### 7.1 动效原则

| 原则 | 值 | 说明 |
|------|-----|------|
| 持续时间 | 200-400ms | 微交互200ms，过渡300ms，场景切换400ms |
| 缓动函数 | ease-out | 进入用 ease-out，退出用 ease-in |
| 弹性 | cubic-bezier(0.34, 1.56, 0.64, 1) | 浇水、开花等"有生命"的动作 |
| 帧率 | 60fps | 所有动画必须流畅 |
| 尊重偏好 | 跟随系统 | 开启"减少动效"时降级为简单过渡 |

### 7.2 核心动效

#### 花朵呼吸动画

```css
/* 2.4秒呼吸周期：吸气1.2s → 屏息0.2s → 呼气1.0s */
@keyframes flower-breathe {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.03); }
}
.flower-living {
  animation: flower-breathe 2.4s ease-in-out infinite;
}
```

#### 浇水动效序列

```
1. 水滴从顶部出现 (0→200ms, ease-out)
2. 水滴落入土壤 (200→400ms, ease-in)
3. 涟漪扩散 (400→700ms, ease-out)
4. 花微微摆动 (+3°/-3°, 3次, 700→1300ms)
5. 粒子飞散 (700→1000ms, 小光点)
6. 数值变化 (同步进行)
```

#### 场景切换过渡

```
页面进入: 从右平移 + 淡入 (300ms, ease-out)
页面退出: 缩小 + 淡出 (200ms, ease-in)
模态弹窗: 从底部滑入 (300ms, cubic-bezier)
```

#### 节气转换动效

```
花园氛围渐变: 8秒缓慢过渡 (仅在用户主动触发时)
  - 背景色渐变
  - 粒子变化 (花瓣→落叶→雪花)
  - 光照角度偏移
  - 音景变化
```
